# GTA VI Countdown ⏳🎮

A sleek, minimalist countdown webpage for GTA VI release date.

## Features

- Live countdown to November 19, 2026
- Google-inspired clean dark theme
- Mobile responsive design
- Shows "GTA VI has been officially released" message when countdown ends

## Usage

Simply open `gta6-countdown.html` in any web browser.

## Tech

- Pure HTML/CSS/JS
- Google Sans font
- No external dependencies
